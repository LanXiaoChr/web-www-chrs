快速开始 ​

简介 ​

Celestial Being 是一个基于 Electron、NodeJS、Vite 和 Vue 的高度还原的界面系统。

安装 ​

运行环境 ​

  • Node.js v20+

开始安装 ​

安装开发环境：

sh
$ npm install

运行 npm run v:dev 来启动一个带有热更新的本地开发服务器：

sh
$ npm run v:dev

运行 npm run e:dev 来启动一个开发模式下的应用：

sh
$ npm run e:dev
